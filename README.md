# kikaku-test-2

このプロジェクトは、技術ブログの GitHub 公開用プロジェクトのテスト用リポジトリです。
GitHubへの公開プロセスの確認等を目的としています。

kikaku-testリポジトリの後にマニュアルの手順確認用として作成したため、kikaku-test-**2**としました。

## 実行方法

### スクリプトの実行
```bash
python3 count.py
```
sample.txtを読み込み、指定した区切り文字（`=====`または`@@SEPARATOR@@`）が現れるまでのテキストを1つのブロックとして扱い、その文字数（スペース、改行コード含む）をカウントして、表示します。
